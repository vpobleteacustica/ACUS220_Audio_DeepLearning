# Referencia e Inspiración: ACUS220 - Acústica Computacional con Python - Clase sobre Audio & Deep Learning

Este directorio contiene material de lectura complementaria para los estudiantes del curso **ACUS220** y que inspiró la clase de Audio & Deep Learning**. Instituto de Acústica, **Universidad Austral de Chile (UACh)**.

---

## Material incluido

### *Practical Deep Learning for Coders (2nd Edition)* – Capítulo 12

**Archivo:** `PracticalDeepLearning2e_Chapter12.pdf`  
**Autores:** Jeremy Howard y Sylvain Gugger  
**Editorial:** O’Reilly Media  
**Año:** 2023  

Este capítulo introduce los fundamentos prácticos del **aprendizaje profundo (Deep Learning)** aplicados a datos de audio, abordando:

- Representación de señales de audio en el dominio tiempo-frecuencia.  
- Espectrogramas, filtros Mel y procesamiento log-mel.  
- Construcción de modelos **CNN** y redes profundas para clasificación de audio.  
- Ejemplos reproducibles con *PyTorch* y *fastai*.  

---

## Propósito educativo

Este material sirve como **inspiración y referencia** para la clase dedicada a **procesamiento de datos de audio de cuatro especies de anfibios y una especie de ave, con aprendizaje profundo**.  
No reemplaza las lecturas oficiales del curso, pero ofrece una visión moderna y aplicada del uso de deep learning en audio.
